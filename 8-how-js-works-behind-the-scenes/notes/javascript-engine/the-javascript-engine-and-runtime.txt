    What is the Javascript engine?

    The program that executes the Javascript code.

    Google's V8 engine powers chrome, and NodeJS.*
    
    
    *NodeJS allows us to make server side applications with Javascript, outside any browser.

    Refer to internet postings for more info on V8 Engine workings: https://cabulous.medium.com/how-v8-javascript-engine-works-5393832d80a7
